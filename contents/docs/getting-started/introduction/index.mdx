## Project Overview

**Shortest** is more than just a testing framework. It's a **complete toolkit** designed for modern testing. Key features include:

- **Natural Language Testing:** Write tests in plain English that AI understands
- **AI-Powered Execution:** Tests are executed by Claude AI
- **GitHub Integration:** Built-in support for GitHub workflows and 2FA
- **Playwright Based:** Reliable browser automation under the hood
