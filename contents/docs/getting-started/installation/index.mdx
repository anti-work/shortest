Setting up Shortest is straightforward and requires minimal configuration. Shortest requires [Node.js](https://nodejs.org/) version 18 or higher for optimal performance.

## Package Installation
To get started with Shortest in your project:
<Stepper>
  <StepperItem title="Step 1: Install the Package">
    Install Shortest using your preferred package manager:

    ```plaintext
    npm install @antiwork/shortest
    ```
  </StepperItem>

  <StepperItem title="Step 2: Configure Environment">
    Create a `.env.local` file in your project root:

    ```plaintext
    ANTHROPIC_API_KEY=your_api_key
    ```
  </StepperItem>

  <StepperItem title="Step 3: Add GitHub Credentials">
    For GitHub authentication tests, add these to your `.env.local`:

    <Note title="Remember">
      GitHub credentials are only needed if you plan to run tests that interact with GitHub.
    </Note>

    ```plaintext
    GITHUB_TOTP_SECRET=your_secret
    GITHUB_USERNAME=your_username
    GITHUB_PASSWORD=your_password
    ```
  </StepperItem>

  <StepperItem title="Step 4: Verify Installation">
    Run the CLI to verify everything is set up:

    ```plaintext
    shortest --help
    ```
  </StepperItem>
</Stepper>

## CLI Commands

Here are the available CLI commands for running tests:

```plaintext
shortest                # Run all tests
shortest login.test.ts  # Run specific test
shortest --headless    # Run in headless mode
shortest --github-code # Generate GitHub 2FA code
```
